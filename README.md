# Iris Flower Classification

This project uses Machine Learning to classify iris flowers into three species:
- Setosa
- Versicolor
- Virginica

Dataset Source: Kaggle

Technologies Used:
- Python
- Scikit-learn
- Pandas
- Matplotlib
- Seaborn

Model:
Logistic Regression

Accuracy:
~95% to 100%
